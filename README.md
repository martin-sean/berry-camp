## Development
This is your typical single page create-react-app application.

Run in development
```sh
$ yarn start
```

Build and deploy yourself to your prefered static hosting site. Netlify and Vercel will handle this for you
```sh
$ yarn build
```

Does nothing... yay!
